# 🎨 Art Gallery Project

## 📌 About the Project
This is my first deployed project — a simple Art Gallery website built using HTML and CSS.  
It displays images in a clean 3x3 grid layout with smooth hover transition effects to enhance user experience.

---

## 🚀 Features
- 🖼️ 3x3 responsive image grid layout
- 🎨 Simple and clean UI design
- ✨ Smooth hover transition effects on images
- 📱 Lightweight and beginner-friendly project
- 🌐 First deployment experience

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  

---

## 📂 How to Run
1. Clone the repository  
2. Open `index.html` in your browser  
3. Enjoy the gallery 🎉
